### Features

- Authentication - Login/Signup
- Rich Text Blogs - Markdown
- Comments

### Requirements

Requires Python 3.5+ and Django 2+

### Installation

```sh
pip install Django
pip install -r requirements.txt

python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```
